# LiveEarthSim 🌍🛰️

A futuristic live Earth simulation project that visualizes our planet in real-time with interactive animations, satellite-style views, and immersive effects. Built for developers, space lovers, and anyone who enjoys exploring Earth from a digital perspective.

---

## ✨ Features

• 🌎 Real-time Earth visualization

• 🛰️ Satellite-inspired live simulation

• 🌌 Smooth animations and interactive UI

• ⚡ Fast and lightweight performance

• 📱 Responsive design for desktop and mobile

• 🎨 Modern futuristic interface

• 🔥 Easy to customize and expand

---

## 🚀 Installation

### Clone the repository:
```bash
git clone https://github.com/surya404root/LiveEarthSim.git
```

### Move into the project folder:
```bash
cd LiveEarthSim
```

### Install dependencies:
```bash
npm install
```

### Run the project:
```bash
npm start
```
